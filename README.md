# ChatClear
> Leert den Chat eines Minecraft Servers

![](ChatClear.png)

## Beispiel für die Nutzung

Falls Extremisten auf deinem Minecraft Server extremistische Sachen schreiben kannst du mit diesen Plugin den Chat leeren
